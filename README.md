## 📖 Overview

This project is part of the Data Science learning journey inspired by the CodeWithHarry course.

The "Coders Of Delhi" project focuses on analyzing a dataset of coders using **pure Python (without external libraries)** to strengthen problem-solving skills and logical thinking.

---

## 🎯 Objectives

* Understand data handling using Python
* Perform data cleaning and preprocessing
* Analyze real-world coder dataset
* Build strong logic without using libraries

---

## 🛠️ Tech Stack

* Language: Python (Standard Library Only)
* Tools: VS Code, Jupyter Notebook, GitHub
